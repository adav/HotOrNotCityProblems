HotOrNotCityProblems
====================

Zorg. A fun game to find out the worries of a city.

---

Requirements

 - python
 - pip
 - django
 - South

---

To run:

 - Use the `git` `bash` tool.
 - Change to the `zorg` directory.
 - Type `./manage.py runserver`
 - The project will now be running at `http://localhost:8000`
